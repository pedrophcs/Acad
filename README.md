# Acad (Sistema de cadastro para academias)

Sistema em **C# (Windows Forms)** com **SQL Server** para cadastro e gestão de alunos.

## ✨ Funcionalidades
- Cadastro, edição e exclusão de alunos
- Vínculo de **Planos** e **Modalidades**
- Listagem com DataGridView (nome, data de nascimento, e-mail, telefone, etc.)

## 🛠️ Tecnologias
- C# (.NET) + Windows Forms
- SQL Server (ADO.NET)
- Git/GitHub

## 🚀 Como executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/pedrophcs/Acad.git

2. Abra a solução no Visual Studio (.sln).

3. Ajuste a connection string no código (ex.: TelaInicial.Conexao).

4. Execute o projeto (F5).

📦 Estrutura (resumo)

- TelaInicial — login e navegação

- Cadastro — tela para inserir alunos

- Registros — listagem, alterar e deletar

- Alterar — edição de dados
